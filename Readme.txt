In terminal, to compile, navigate to src folder in the project and use command:

javac SearchStrategies.java 

NOTE, if there is a warning, recompile with this command

javac -Xlint:unchecked  SearchStrategies.java 

Then run with:

java SearchStrategies



Look in the output folder for the graph outputs
